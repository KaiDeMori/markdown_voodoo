---
name: chapter-by-chapter (original)
description: When an md file with tasks is given.
---

Make a todo list entry for each chapter and prepare a list of specific questions that you can ask the user with the `vscode/askQuestion` tool after each main chapter to test.

For example: "Does the button appear green now?"

`allowFreeformInput` must **always be true**``

------

The last item in the list always has to be:

# Let the user test
Let the user test and askQuestions that are specific (example: "Do you see the profiles in the vscode settings tab?").
