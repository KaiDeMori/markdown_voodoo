# Markdown Voodoo

![Markdown Voodoo banner](assets/markdown_voodoo_banner.avif)

A small spellbook of prompts, agents, instructions, and workflow charms for shaping AI coding assistants inside VS Code.

This repo is not a framework, not a product, and definitely not a cathedral. It is a workbench: practical little markdown artifacts that help an AI assistant plan better, ask better questions, write cleaner commits, follow project conventions, and generally behave less like a random autocomplete cloud.

## What's Inside

- `agents/`  
  Custom agent modes for different working styles.

- `instructions/`  
  Reusable coding and collaboration rules.

- `prompts/`  
  Focused prompt workflows for planning, implementation, investigation, and writing.

- `commit_instructions/`  
  Commit message guidance for cleaner project history.

- `skills/`  
  Larger workflow packages for repeatable project rituals.

## Philosophy

Markdown Voodoo is built around a simple idea:

Good AI behavior is not magic.  
But a little ritual helps.

The files here are meant to be readable, tweakable, and personal. Copy what helps, rename what annoys you, delete what does not fit, and keep sharpening the parts that make your own workflow feel smoother.
