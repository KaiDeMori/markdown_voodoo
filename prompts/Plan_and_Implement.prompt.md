---
name: Plan_and_Implement
description: Instructions for planning and implementing code changes.
---

1. Plan the change
2. Divide the change into manageable steps
3. Add a todo list item for each step.
4. The final todo must always be "Let the user test".
5. Implement the steps one by one
6. Build and test
7. "Let the user test" — This means: Let the user test and askQuestions that are specific (example: "Do you see the new button?").
